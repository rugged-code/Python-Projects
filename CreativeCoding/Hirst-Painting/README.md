# HirstPainting 🎨

A generative art project inspired by the spot paintings of Damien Hirst using Python Turtle graphics.

## Preview

Add your generated artwork screenshot here.

## Features

* Randomized color generation
* Grid-based dot painting
* Turtle graphics rendering
* RGB color extraction using Colorgram

## Technologies Used

* Python
* Turtle Graphics
* Colorgram
* Random Module

## How to Run

1. Clone the repository
2. Navigate to the project folder
3. Run:

```bash
python main.py
```
