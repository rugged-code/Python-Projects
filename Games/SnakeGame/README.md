# Snake Game 🐍

A classic Snake Game built using Python’s built-in `turtle` graphics module.
Control the snake, eat food, grow longer, and avoid colliding with the walls or your own tail.

---

## Features

* Smooth snake movement
* Random food spawning
* Score tracking system
* Collision detection
* Dynamic speed increase as the score grows
* Keyboard controls
* Modular Object-Oriented Programming (OOP) structure

## How to Run

### 1. Clone the Repository

```bash
git clone <your-repository-link>
```

### 2. Navigate to the Project Folder

```bash
cd SnakeGame
```

### 3. Run the Game

```bash
python main.py
```

---

## Game Rules

* Eat the food to increase your score.
* Every food item makes the snake longer.
* The snake speed gradually increases as the score increases.
* Avoid hitting the walls.
* Avoid colliding with your own tail.
