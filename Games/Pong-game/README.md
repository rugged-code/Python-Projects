# Pong Game 🎮

A classic two-player Pong game built using Python’s built-in `turtle` module.
Control the paddles, bounce the ball, and compete against your friend in this retro arcade-style game.

---

## Features

* Two-player gameplay
* Smooth paddle movement
* Ball collision mechanics
* Live score tracking
* Ball speed increases after each paddle hit
* Simple and clean OOP-based structure

---

## Controls

| Player       | Move Up   | Move Down |
| ------------ | --------- | --------- |
| Left Paddle  | `W`       | `S`       |
| Right Paddle | `↑ Arrow` | `↓ Arrow` |

## How to Run

1. Make sure Python is installed.
2. Clone the repository:

```bash
git clone <your-repo-link>
```

3. Open the project folder:

```bash
cd pong-game
```

4. Run the game:

```bash
python main.py
```

## Game Mechanics

* The ball bounces off walls and paddles.
* Each paddle hit increases the ball speed.
* Missing the ball gives the opponent a point.
* Score updates automatically on screen.
